# Masterboard Home

Site institucional do Masterboard em Astro.

## Estrutura

- `site/`: aplicação Astro, componentes, dados, estilos e assets públicos.
- `CHANGELOG.md`: histórico das mudanças feitas no projeto.

## Desenvolvimento

```bash
cd site
npm install
npm run dev
```

## Build

```bash
cd site
npm run build
```

## O que não vai para o GitHub

- `site/node_modules/`
- `site/dist/`
- `site/.astro/`
- arquivos `.env`
- backups, exports e bancos `.sql`
- credenciais
